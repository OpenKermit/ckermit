# C-Kermit Scripting

C-Kermit has a robust and powerful scripting language built in.

As noted on the [Kermit Documentation](https://www.openkermit.org/doc/) page,
Kermit has prized compatibility for many years.  Resources from the older Kermit
Project are still online and are perfect to introduce you to the C-Kermit
scripting language.

Here are some places to consult:

- [The C-Kermit Tutor section on Scripts](https://www.kermitproject.org/ckututor.html#kerbang)
- [The C-Kermit Scripting Tutorial and Library](https://www.kermitproject.org/ckscripts.html)
